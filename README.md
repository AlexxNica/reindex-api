![Build status](https://circleci.com/gh/freiksenet/reindex-api.svg?style=shield&circle-token=080d24db0ad712462742bb77cd91a316c2267e46)

### Installation

- Install RethinkDB
- Install dependencies

  ```
  npm install
  ```
- Run tests

  ```
  npm test
  ```
- Create a test app

  ```
  npm run create-app localhost.reindexio.com
  ```
- Run server

  ```
  npm start
  ```
